# 🎨 Mogle - 새로운 UI/UX 디자인 가이드

## 브랜드 정보

**회사명**: Mogle
**슬로건**: 개인 AI 코치
**색상**: 보라색 그래디언트 (#667eea → #764ba2)

---

## 📐 새로운 레이아웃 아키텍처

### 전체 구조

```
┌─────────────────────────────────────────────────────┐
│                  HEADER (고정)                      │
├──────────────────┬──────────────────────────────────┤
│                  │                                  │
│   SIDEBAR        │        MAIN CONTENT AREA         │
│  (좌측 고정)     │       (그리드 레이아웃)         │
│                  │                                  │
│  • 대시보드       │  ┌─ 카드 ┬─ 카드 ─────┐        │
│  • 감정 기록      │  │      │            │        │
│  • 목표 관리      │  ├──────┼──────────┤        │
│  • 분석          │  │    카드 (풀 너비)  │        │
│                  │  └──────────────────┘        │
└──────────────────┴──────────────────────────────────┘
```

---

## 🎯 주요 특징

### 1️⃣ 사이드바 (Sidebar)

**위치**: 왼쪽 고정, 항상 표시
**너비**: 280px
**배경**: 보라색 그래디언트

```
┌────────────────────┐
│ 🧠 Mogle          │  ← 로고 + 회사명
│ 개인 AI 코치        │  ← 슬로건
├────────────────────┤
│                    │
│ 📊 대시보드         │  ← 활성 (파란 테두리)
│ 😊 감정 기록        │
│ 🎯 목표 관리        │
│ 📈 분석           │
│                    │
├────────────────────┤
│  Made by Mogle    │  ← 푸터
└────────────────────┘
```

**네비게이션 항목**:

- **대시보드**: AI 피드백 + 그래프 + 목표 정리
- **감정 기록**: 새 감정 추가
- **목표 관리**: 목표 CRUD
- **분석**: 상세 분석 및 통계

### 2️⃣ 헤더 (Header)

**위치**: 상단 고정
**높이**: 80px
**배경**: 흰색
**음영**: 미묘한 그림자

```
☰ | 나만의 AI 코치
    당신의 감정과 목표를 함께 추적해보세요
```

**요소**:

- 메뉴 토글 버튼 (☰) - 모바일에서만 표시
- 페이지 제목
- 페이지 설명

### 3️⃣ 메인 콘텐츠 (Main Content)

**레이아웃**: CSS 그리드
**간격**: 1.5rem
**컨테이너 너비**: 반응형 (400px 이상)

#### 대시보드 뷰

```
┌─────────────────────────────────────┐
│  AI 피드백 (풀 너비)               │
├──────────────────┬──────────────────┤
│  감정 그래프     │  목표 관리       │
│  (2열 or 1열)   │  (2열 or 1열)   │
└──────────────────┴──────────────────┘
```

## 📱 반응형 디자인

### 데스크톱 (> 1024px)

- 사이드바: 280px 고정
- 콘텐츠: 2열 그리드
- 헤더: 전체 정보 표시

### 태블릿 (768px - 1024px)

- 사이드바: 상단으로 변경
- 콘텐츠: 1-2열 그리드
- 헤더: 축약된 정보

### 모바일 (< 768px)

- 사이드바: 하단 네비게이션 바로 변경
- 콘텐츠: 1열 그리드
- 헤더: 최소 정보

### 초소형 모바일 (< 480px)

- 사이드바: 하단 탭 바
- 콘텐츠: 전체 너비
- 헤더: 아이콘만 표시

```

├─ AI 피드백 카드 (클릭 불가, 읽기만)

   ├─ 감정 선택
   ├─ 강도 조절

   ├─ 노트 추가
   └─ 저장

목표 관리
├─ 목표 목록 보기
├─ 새 목표 추가
└─ 진행도 업데이트

분석
├─ 감정 그래프 (대형)
└─ AI 인사이트
```

## 🎪 컴포넌트 스타일

### 카드 (Card)

```
┌─────────────────┐
│  제목           │
├─────────────────┤
│                 │
│  컨텐츠         │
│                 │
└─────────────────┘

스타일:
- 배경: 흰색
- 그림자: 0 2px 8px rgba(0,0,0,0.08)
- 모서리: 12px 반지름
- 호버: 그림자 증가
```

### 버튼

```
[텍스트]

스타일:
- 기본: 보라색 배경
- 호버: 진보라 배경
- 활성: 밝은 테두리
```

### 입력 필드

```
┌───────────────────┐
│ 플레이스홀더      │
└───────────────────┘

스타일:
- 테두리: 1px solid #e0e0e0
- 호버: 테두리 색상 진해짐
- 포커스: 보라색 테두리
```

## 🚀 구현된 기능

✅ **사이드바 네비게이션**

- 4가지 메인 섹션
- 활성 상태 표시
- 호버 효과

✅ **고정 헤더**

- 페이지 제목
- 메뉴 토글 버튼 (모바일)

✅ **그리드 레이아웃**

- 반응형 2열 그리드
- 카드 기반 구성

✅ **모바일 최적화**

- 반응형 사이드바
- 하단 탭 네비게이션 (모바일)

✅ **Mogle 브랜드**

- 로고 + 회사명
- 슬로건 표시
