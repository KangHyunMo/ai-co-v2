## 1️⃣ 감정 추적 섹션 (Emotion Tracker)

### 화면 구성

```
┌─────────────────────────────────────┐
│  🎯 오늘의 감정을 기록해주세요      │
├─────────────────────────────────────┤
│ [😢] [😔] [😐] [😊] [😄]          │  ← 감정 선택 버튼
│   매우슬  슬  보  행  매우행         │
├─────────────────────────────────────┤
│ 강도 (1-10)                         │
│ ━━━━━━●━━━ 5/10                   │  ← 강도 슬라이더
├─────────────────────────────────────┤
│ 기분 상태 (예: 피곤함, 스트레스)   │
│ [________________] 입력 필드        │
├─────────────────────────────────────┤
│ 추가 노트                           │
│ [________________________]           │  ← 여러 줄 입력
│ [________________________]           │
├─────────────────────────────────────┤
│ [감정 기록하기 ✨]                 │  ← 저장 버튼
└─────────────────────────────────────┘
```

### 기능 설명

- **감정 버튼**: 클릭하면 선택 상태 변경 (테두리 밝음)
- **강도 슬라이더**: 마우스로 드래그하여 1-10 값 설정
- **텍스트 필드**: 기분 상태와 노트 작성
- **저장 버튼**: 모든 데이터 자동 저장 (alert로 확인)

### 색상 스키마

- 배경: 보라색 그래디언트 (#667eea → #764ba2)
- 버튼: 흰색 테두리, 투명 배경
- 활성: 밝은 테두리, 약간 밝아진 배경

## 2️⃣ 목표 모니터링 섹션 (Goal Monitor)

### 화면 구성

```
┌─────────────────────────────────────┐
│  🎯 목표 모니터링                  │
├─────────────────────────────────────┤
│ [+ 새 목표 추가]                   │
├─────────────────────────────────────┤
│ ┌─ 목표 1 ──────────────────────┐ │
│ │ 제목: 매일 운동하기 [건강]    │ │
│ │ 설명: 건강한 몸을 위해...     │ │
│ │ 진행도: [=====────] 50%       │ │
│ │ [−5%] [+5%]                   │ │
│ │ 남은 일수: 15 ✅ 완료!       │ │
│ └─────────────────────────────────┘ │
│ ┌─ 목표 2 ──────────────────────┐ │
│ │ 제목: 책 읽기 [학습]          │ │
│ │ 설명: 월 1권 읽기             │ │
│ │ 진행도: [════───────] 30%     │ │
│ │ [−5%] [+5%]                   │ │
│ │ 남은 일수: 20                 │ │
│ └─────────────────────────────────┘ │
└─────────────────────────────────────┘
```

### 기능 설명

- **새 목표 추가**: 클릭하면 입력 폼 표시
- **진행도 바**: 시각적 진행 상황 표현
- **±5% 버튼**: 빠른 진행도 조정
- **자동 완료**: 100% 도달 시 완료 상태로 변경

### 카테고리별 색상

- 건강: 초록색
- 업무: 파란색
- 개인: 보라색
- 관계: 분홍색
- 학습: 주황색

```
┌─────────────────────────────────────┐
│  📊 감정 변화 그래프               │
├─────────────────────────────────────┤
│ ┌─ 통계 ──────────────────────────┐ │
│ │ 총 기록 수: 15    │ 평균 감정: 3.5/5 │
│ └──────────────────────────────────┘ │
├─────────────────────────────────────┤
│      5 ╱╲           ╱╲            │
│        ╱  ╲        ╱  ╲           │
│      3 ╱    ╲____╱    ╲____       │
│        01   03   05   07   09  11  │
│       날짜                         │
│ ★ 평균감정점수  ★ 강도             │
├─────────────────────────────────────┤
│ 감정 분포                           │
│ 매우행복: ███████░ 7개             │
│ 행복:    ████░░░░░░ 4개           │
│ 보통:    ██░░░░░░░░ 2개           │
│ 슬픔:    █░░░░░░░░░ 1개           │
│ 매우슬프: 1개                       │
└─────────────────────────────────────┘
```

### 상호작용

- **차트 마우스 오버**: 도구팁 표시
  - 날짜
  - 평균 감정 점수
  - 강도
  - 기분 상태
- **범례**: 라인 클릭으로 표시/숨김 (가능)

## 4️⃣ AI 코치 피드백 (AI Feedback)

### 화면 구성

```
┌─────────────────────────────────────┐
│  🤖 AI 코치 피드백                │
├─────────────────────────────────────┤
│ ┌─ 피드백 카드 ──────────────────┐ │
│ │ "요즘 기분이 좋아졌어요!"    │ │
│ │                              │ │
│ │ [💡 인사이트]               │ │
│ └──────────────────────────────┘ │
├─────────────────────────────────────┤
│ 📈 감정 트렌드                      │
│ ┌──────────────────────────────┐ │
│ │ 당신의 기분이 좋아지고       │ │
│ │ 있어요! 🌱                   │ │
│ └──────────────────────────────┘ │
├─────────────────────────────────────┤
│ 💭 인사이트                         │
│ ┌──────────────────────────────┐ │
│ │ ✓ 최근 좋은 기분이 많았어요 │ │
│ │ ✓ 기분이 점점 좋아지고 있어요│ │
│ └──────────────────────────────┘ │
├─────────────────────────────────────┤
│ 🎯 추천사항                         │
│ ┌──────────────────────────────┐ │
│ │ ✓ 현재 기분을 유지해보세요  │ │
│ │ ✓ 긍정적인 활동을 계속하세요│ │
│ └──────────────────────────────┘ │
├─────────────────────────────────────┤
│ 😊 좋은 기분 순간                   │
│ [행복함] [신남] [편함] [감사함]    │
├─────────────────────────────────────┤
│ 😔 힘든 순간                        │
│ [피곤함] [스트레스]                │
├─────────────────────────────────────┤
│ [분석 새로고침 🔄]                 │
└─────────────────────────────────────┘
```

### 피드백 타입

1. **💡 인사이트**: 데이터 기반 분석
2. **💪 격려**: 긍정적 피드백
3. **💭 제안**: 실행 가능한 조언
4. **⚠️ 주의**: 주의가 필요한 상황

### 태그 스타일

- **긍정 태그**: 녹색 배경, 흰색 텍스트
- **부정 태그**: 빨간색 배경, 흰색 텍스트

## 📱 반응형 디자인

### 모바일 (< 480px)

- 한 줄 레이아웃
- 더 큰 버튼
- 축소된 차트

### 태블릿 (480-768px)

- 2열 레이아웃
- 조정된 폰트 크기
- 최적화된 간격

### 데스크톱 (> 768px)

- 넉넉한 레이아웃
- 풀 사이즈 차트
- 멀티 컬럼 카드

## ⌨️ 키보드 단축키 (향후 추가 예정)

