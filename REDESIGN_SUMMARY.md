
```

[콘텐츠 - 1열]
[하단 아이콘 탭]
```

## 🎯 주요 개선사항

### 1. 네비게이션 개선 ⭐

- **이전**: 페이지 로드 시마다 새로고침
- **새로운**: 탭만 변경 (빠름)

### 2. 시각적 계층 구조 ⭐

- **이전**: 모든 섹션이 동등하게 표시
- **새로운**: 사이드바 + 콘텐츠 명확 분리

### 3. 브랜드 강화 ⭐

- **이전**: "나만의 AI 코치"만 표시
- **새로운**: "Mogle" 회사명 강조

### 4. 공간 활용 ⭐

- **이전**: 세로 스크롤 무한
- **새로운**: 그리드 레이아웃으로 효율적

### 5. 전문성 ⭐

- **이전**: 단순한 느낌
- **새로운**: 현대적 대시보드

## 🎮 사용 흐름

### 시나리오 1: 일일 감정 기록

```
1. 사이드바에서 "😊 감정 기록" 클릭
2. 감정 입력 폼 표시
3. 감정 선택 → 강도 조절 → 기분 설명 → 저장
4. 완료!
```

### 시나리오 2: 목표 진행도 업데이트

```
1. 사이드바에서 "🎯 목표 관리" 클릭
2. 목표 목록 표시
3. 진행도 ±5% 조절
4. 자동 저장
```

### 시나리오 3: 분석 리뷰

```
1. 사이드바에서 "📈 분석" 클릭
2. 감정 그래프 상세 표시
3. AI 인사이트 읽기
4. 패턴 파악
```

### 시나리오 4: 대시보드 확인

```
1. 사이드바에서 "📊 대시보드" 클릭 (기본)
2. AI 피드백 확인
3. 감정 그래프 스캔
4. 목표 진행률 체크
```

## 🛠️ 기술 구현

### 상태 관리

```javascript
const [activeSection, setActiveSection] = useState("dashboard")
// dashboard, emotion, goals, analytics
```

### 조건부 렌더링

```javascript
{
  activeSection === "dashboard" && <DashboardView />
}
{
  activeSection === "emotion" && <EmotionView />
}
{
  activeSection === "goals" && <GoalsView />
}
{
  activeSection === "analytics" && <AnalyticsView />
}
```

### 반응형 미디어 쿼리

```css
/* 데스크톱 */
.sidebar {
  width: 280px;
  position: fixed;
}

/* 태블릿 */
@media (max-width: 768px) {
  .sidebar {
    position: fixed;
    top: 0;
    height: auto;
  }
}

/* 모바일 */
@media (max-width: 480px) {
  .sidebar {
    position: fixed;
    bottom: 0;
    flex-direction: row;
  }
}
```
